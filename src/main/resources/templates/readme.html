Welcome to my small project.
You can play 2 games, add/delete ships from database or get information about various ship types or manufacturers.
If you click on Types(Escorts, Cruisers, Battleships) or Manufacturers(Imperial Navy, Space Marines), it will redirect to respective web page, where you can get more info about it.
You can add new ships by clicking to Create new Ship button.
You can delete existing ships by clicking to individual Ship titles.
You can play games by clicking on Play game, and Play another game.
Project is still under construction.
